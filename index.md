# 개인정보처리방침

**도제 라이프** (이하 "앱")은 경기자동차과학고등학교(이하 "학교")가 도제교육 참여 학생 및 교사의 출결 관리, 공지사항 전달, 일정 관리, 소통을 목적으로 운영합니다.

본 개인정보처리방침은 앱이 수집하는 개인정보의 항목, 수집 목적, 보유 기간 및 이용자의 권리에 대해 안내합니다.

---

## 1. 개인정보 수집 항목 및 목적

### 1-1. 필수 수집 항목

| 항목 | 수집 목적 |
|------|-----------|
| Google 계정 이메일 | 로그인 인증, 학생/교사 역할 구분 |
| 이름 | 출결 기록, 공지사항, 채팅 표시 |
| 전화번호 | 비상 연락, 교사-학생 간 연락 |
| 생년월일 | 학생 신원 확인 |
| 학년 · 반 | 출근 일정 및 공지사항 대상 구분 |
| 소속 기업 | 출근 장소 안내, 출결 관리 |
| Firebase 사용자 UID | 계정 식별 및 데이터 연결 |

### 1-2. 서비스 이용 과정에서 생성·수집되는 정보

| 항목 | 수집 목적 |
|------|-----------|
| 출결 인증 사진 | OCR로 입·퇴실 시간 추출, 출결 증빙 보관 |
| 입·퇴실 시간 | 출결 현황 관리, 지각 여부 판단 |
| 위치 정보 | 날씨 정보 표시 (출결 인증과 무관) |
| FCM 기기 토큰 | 푸시 알림 발송 |
| 채팅 메시지 내용 | 교사-학생 간 1:1 소통 |
| 개인 일정 | 학생 개인 일정 관리 |
| 앱 접속 로그 | 관리자 활동 기록 |

## 2. 개인정보 수집 방법

- Google 로그인을 통한 직접 제공
- 회원가입 시 이용자가 직접 입력
- 앱 사용 과정에서 자동 생성

---

## 3. 개인정보 보유 및 이용 기간

| 구분 | 보유 기간 |
|------|-----------|
| 재학생 계정 및 관련 데이터 | 졸업일(입학년도 + 3년) 이후 자동 삭제 |
| 출결 기록 | 해당 학생 계정 삭제 시 함께 삭제 |
| 채팅 메시지 | 해당 채팅방 삭제 시 함께 삭제 |
| 출결 인증 사진 | Firebase Storage에 보관, 계정 삭제 시 함께 삭제 |
| 관리자 활동 로그 | 학교 운영 기간 동안 보관 |

> 단, 관련 법령에 따라 보존이 필요한 경우 해당 기간 동안 보관됩니다.

---

## 4. 개인정보의 제3자 제공

앱은 수집한 개인정보를 외부에 제공하지 않습니다. 다만 아래의 경우는 예외입니다.

- 이용자가 사전에 동의한 경우
- 법령에 따라 수사기관 등이 요구하는 경우

---

## 5. 개인정보 처리 위탁

앱은 서비스 운영을 위해 아래 업체에 개인정보 처리를 위탁합니다.

| 수탁업체 | 위탁 내용 |
|----------|-----------|
| Google LLC (Firebase) | 데이터베이스(Firestore), 인증(Auth), 파일 저장(Storage), 푸시 알림(FCM) |
| Google LLC (ML Kit) | 출결 사진 OCR 처리 (온디바이스, 서버 미전송) |

> Firebase 데이터는 Google의 개인정보처리방침을 따릅니다.  
> https://policies.google.com/privacy

---

## 6. 이용자의 권리

이용자(학생·교사)는 언제든지 다음의 권리를 행사할 수 있습니다.

- **열람권:** 본인의 개인정보 처리 현황 조회
- **정정권:** 잘못된 개인정보 수정 요청
- **삭제권:** 계정 및 개인정보 삭제 요청
- **처리 정지권:** 개인정보 처리 정지 요청

권리 행사는 담당 교사 또는 학교 관리자를 통해 요청할 수 있습니다.

---

## 7. 위치정보 이용

앱은 홈 화면의 날씨 정보 표시를 위해 기기의 위치 정보를 수집합니다.

- 수집 방식: 앱 사용 중에만 수집 (백그라운드 미수집)
- 위치 정보는 외부 서버에 저장되지 않으며, 날씨 API 호출 후 즉시 파기됩니다.
- 위치 정보 제공을 거부할 경우 날씨 기능을 사용할 수 없으나, 그 외 서비스 이용에는 제한이 없습니다.

---

## 8. 카메라 및 사진 접근

앱은 출결 인증을 위해 갤러리 접근 권한을 요청합니다.

- 선택한 사진은 OCR(문자 인식)을 통해 입·퇴실 시간을 추출하는 데 사용됩니다.
- OCR은 기기 내에서 처리되며(온디바이스), 원본 사진은 Firebase Storage에 증빙 목적으로 저장됩니다.

---

## 9. 푸시 알림

앱은 다음의 경우 푸시 알림을 발송합니다.

- 교사가 새 공지사항을 등록한 경우 (대상 학년·반에 해당하는 학생에게만 발송)
- 채팅 메시지를 수신한 경우

알림 수신을 원하지 않을 경우 기기의 설정에서 알림을 비활성화할 수 있습니다.
