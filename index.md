# 개인정보처리방침

**GHAS 도제**(이하 "앱")은 경기자동차과학고등학교(이하 "학교")가 도제교육 참여 학생 및 교사의 출결 관리, 공지사항 전달, 일정 관리, 소통을 목적으로 운영하는 서비스입니다.

본 개인정보처리방침은 앱이 처리하는 개인정보의 항목, 처리 목적, 보유 기간, 파기 방법, 이용자의 권리 및 개인정보 보호 조치에 대해 안내합니다.

- 시행일: 2026년 4월 20일
- 최종 수정일: 2026년 5월 6일

## 4. 개인정보 파기 절차 및 방법

앱은 보유 기간 경과, 처리 목적 달성, 계정 삭제 요청 등 파기 사유가 발생하면 아래 절차에 따라 개인정보를 파기합니다.

1. 파기 대상 확인: 보유 기간이 경과했거나 삭제 요청이 접수된 계정 및 관련 데이터를 확인합니다.
2. 데이터 삭제: Firestore 문서, Firebase Storage 파일, FCM 토큰 등 관련 데이터를 삭제합니다.
3. 계정 삭제: Firebase Auth 계정 삭제가 필요한 경우 관리자 권한을 가진 서버 기능을 통해 삭제 처리합니다.
4. 로그 기록: 관리자 또는 시스템에 의한 주요 삭제 작업은 오남용 방지와 감사 목적을 위해 관리자 활동 로그에 기록될 수 있습니다.

전자적 파일 형태의 개인정보는 복구가 어렵도록 삭제하며, 종이 문서로 출력된 개인정보가 있는 경우 분쇄 또는 파쇄합니다.

## 6. 개인정보 처리 위탁 및 외부 서비스 이용

앱은 안정적인 서비스 운영을 위해 아래 서비스를 이용합니다.

| 수탁업체 또는 서비스 | 이용 목적 | 처리되는 정보 |
|----------------------|-----------|---------------|
| Google LLC / Firebase Authentication | Google 로그인, 사용자 인증 | 이메일, UID, 인증 정보 |
| Google LLC / Firebase Firestore | 앱 데이터베이스 운영 | 계정 정보, 출결 기록, 공지사항, 일정, 채팅, 관리자 로그 |
| Google LLC / Firebase Storage | 출결 인증 사진 및 채팅 이미지 저장 | 이용자가 선택한 이미지 파일 |
| Google LLC / Firebase Cloud Messaging | 푸시 알림 발송 | FCM 기기 토큰, 알림 데이터 |
| Google LLC / ML Kit Text Recognition | 출결 사진 OCR 처리 | 사용자가 선택한 사진의 문자 인식 결과 |
| OpenMeteo GmbH / Open-Meteo API | 홈 화면 날씨 정보 조회 | 현재 위치 좌표, API 요청 정보 |

ML Kit OCR 처리는 기기 내에서 수행되며, OCR 처리를 위해 사진이 Google OCR 서버로 전송되지는 않습니다. 다만 출결 증빙 확인을 위해 사용자가 선택한 출결 인증 사진은 Firebase Storage에 업로드될 수 있습니다.

## 7. 개인정보의 국외 이전

앱은 Firebase 및 날씨 API 등 국외 서비스를 이용하므로 서비스 제공 과정에서 개인정보가 국외로 이전될 수 있습니다.

| 이전받는 자 | 이전 국가 | 이전 항목 | 이전 목적 | 이전 일시 및 방법 | 보유 기간 |
|-------------|-----------|-----------|-----------|-------------------|-----------|
| Google LLC | 미국 등 Google Cloud 데이터센터 소재 국가 | 이메일, UID, 계정 정보, 출결 기록, 공지사항, 일정, 채팅, 사진, FCM 토큰 등 | 인증, 데이터 저장, 파일 저장, 푸시 알림 | 앱 이용 시 암호화된 네트워크 전송 | 본 방침의 보유 기간에 따름 |
| OpenMeteo GmbH | 스위스 등 서비스 운영 국가 | 현재 위치 좌표, API 요청 정보 | 날씨 정보 조회 | 날씨 정보 요청 시 HTTPS 전송 | 앱에는 저장하지 않으며, Open-Meteo는 API 사용 로그를 일정 기간 보관할 수 있음 |

이용자는 위치 권한을 거부하여 Open-Meteo API로 현재 위치 좌표가 전송되지 않도록 할 수 있습니다. 이 경우 날씨 기능은 제한되지만, 출결·공지·채팅 등 다른 기능 이용에는 제한이 없습니다.

## 8. 이용자 및 법정대리인의 권리와 행사 방법

이용자(학생·교사·관리자) 및 관련 법령상 법정대리인은 언제든지 다음 권리를 행사할 수 있습니다.

- 개인정보 열람 요청
- 개인정보 정정 요청
- 개인정보 삭제 요청
- 개인정보 처리 정지 요청
- 계정 삭제 요청

권리 행사는 앱 내 담당 교사 또는 학교 관리자에게 요청하거나, 본 방침의 개인정보 보호 담당자 연락처로 요청할 수 있습니다. 학교는 본인 확인 후 관련 법령과 학교 운영 기준에 따라 지체 없이 처리합니다.

계정 삭제를 요청하면 법령 또는 학교 운영상 보관이 필요한 정보를 제외하고 계정 정보와 관련 데이터를 삭제하거나 익명화합니다. 앱 외부에서 계정 삭제를 요청하려는 경우 본 방침의 개인정보 보호 담당자 이메일로 요청할 수 있습니다.

## 9. 위치정보 이용

앱은 홈 화면의 현재 날씨 표시를 위해 기기의 위치 정보를 사용합니다.

- 수집 방식: 앱 사용 중 위치 권한을 허용한 경우에만 수집
- 이용 목적: 현재 위치 기반 날씨 정보 표시
- 외부 전송: 날씨 정보 조회를 위해 위도·경도 좌표가 Open-Meteo API로 전송될 수 있음
- 저장 여부: 앱 서버 또는 Firebase에 위치 정보를 저장하지 않음
- 거부권: 위치 권한을 거부해도 날씨 기능 외 서비스 이용에는 제한 없음

위치 정보는 출결 인증, 출근 장소 판정, 근태 평가 목적으로 사용하지 않습니다.

## 10. 사진 접근

앱은 출결 인증과 채팅 이미지 전송을 위해 사진 접근 권한을 요청할 수 있습니다.

- 출결 인증: 사용자가 선택한 사진에서 OCR로 입·퇴실 시간을 추출합니다.
- 증빙 보관: 출결 인증 사진은 Firebase Storage에 업로드될 수 있으며, 업로드 후 48시간이 지나면 자동 삭제됩니다.
- 채팅 이미지: 사용자가 채팅방에서 선택한 이미지는 해당 채팅방 참여자에게 표시됩니다.
- 선택권: 사용자가 선택하지 않은 사진에는 접근하지 않습니다.

출결 인증 사진에는 얼굴, 이름표, 근무지 화면 등 개인정보가 포함될 수 있으므로 출결 증빙 및 확인 목적 외에는 사용하지 않습니다.

---

## 11. 푸시 알림

앱은 다음의 경우 푸시 알림을 발송합니다.

- 교사 또는 관리자가 새 공지사항을 등록한 경우
- 채팅 메시지를 수신한 경우
- 학교 운영상 필요한 주요 안내가 있는 경우

채팅 알림은 사용자가 해당 대화방을 이미 열고 있는 경우 표시되지 않을 수 있습니다. 공지사항은 중요 안내이므로 앱에서 공지사항 탭을 보고 있어도 알림이 표시될 수 있습니다.

알림 수신을 원하지 않는 경우 기기 설정에서 앱 알림을 비활성화할 수 있습니다.

## 12. 개인정보의 안전성 확보 조치

학교와 앱은 개인정보 보호를 위해 다음과 같은 조치를 적용합니다.

- Firebase Authentication을 통한 사용자 인증
- Firestore Security Rules를 통한 역할 기반 접근 제어
- HTTPS/TLS 기반 암호화 통신
- Firebase Storage 접근 권한 제한
- 학생, 교사, 관리자 역할별 화면 및 기능 접근 제한
- 관리자 주요 작업 로그 기록
- 불필요한 출결 인증 사진 자동 삭제
- 테스트용 숨김 계정 등 일반 사용자 목록 노출 제한

## 13. 자동 수집 장치 및 광고 추적

앱은 웹 쿠키를 사용하지 않으며, 광고 목적의 추적 SDK 또는 광고 식별자를 사용하지 않습니다.

Firebase, 기기 운영체제, 앱 스토어 또는 외부 API 제공자가 서비스 안정성, 보안, 오류 분석을 위해 별도의 기술 정보를 처리할 수 있으며, 해당 처리는 각 제공자의 개인정보처리방침을 따릅니다.

## 14. 만 14세 미만 아동에 관한 사항

앱은 경기자동차과학고등학교 도제교육 참여 학생 및 교사를 대상으로 하며, 만 14세 미만 아동을 주된 대상으로 하지 않습니다. 만 14세 미만 이용자의 개인정보 처리가 필요한 경우 법정대리인의 동의 등 관련 법령상 필요한 절차를 따릅니다.
